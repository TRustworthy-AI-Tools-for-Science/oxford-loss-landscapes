# Documentation

This directory contains documentation for the Oxford Loss Landscapes package.

## Structure

- `README.md` - This file, overview of documentation

## Building Documentation

To build Sphinx documentation (when implemented):

```bash
pip install -e .[docs]
cd docs/
make html
```
